# [주제] : 요리 레시피 프로그램

## [CRUD 기능]
> C : 요리 이름과 설명 레시피 \n
> R : 모든 요리 나열과 선택에 따라 자세히 나오는 레시피
> U : 요리 이름 및 레시피 설명 내용 수정
> D : 요리 레시피 삭제

## [파일 r/w]
> Lode : .txt 파일 읽어 오기
> Save : 수정된 데이터값 다시 .txt 파일에 save 하기

## [요리 검색 3가지 방법]
> 1. 이름으로 검색 ("순두부 찌개")
> 2. 카테고리로 검색 (#한식)

## [랜덤 요리 추천]
> 뭐먹을지 고민될 때, 랜덤으로 요리를 추천해 줌
